# Halabalusa 

Landing page y plataforma para **Halabalusa**, movimiento underground que fusiona **identidad, cultura y estética**.  
El proyecto busca documentar, difundir y conectar la historia, las batallas, las noticias y la comunidad de Halabalusa, con un enfoque visual claro y funcional.

---

## Tecnologías utilizadas

- **Frontend:** React 19.1.1 + React DOM 19.1.1  
- **Routing:** React Router DOM 7.1.1  
- **Formularios:** React Hook Form 7.62.0 + Zod 4.0.17  
- **Estilos:** Tailwind CSS 3.4.17 (con Flexbox y Grid para responsividad)  
- **Build Tool:** Vite 6.0.5  
- **Lenguaje:** TypeScript 5.7.2  
- **HTTP Client:** Axios 1.11.0  
- **Testing/Mocks:** MSW 2.10.4  
- **Linting & Formatting:** ESLint 9.17.0 + Prettier 3.4.2  
- **Plugins:** @vitejs/plugin-react-swc, @typescript-eslint, eslint-plugin-react-hooks, eslint-plugin-react-refresh  

---

## Estructura del proyecto

- `src/components/Navbar.tsx` → Barra de navegación fija con logo y menú responsive.  
- `public/images/halasinfondo.png` → Logo oficial de Halabalusa (versión sin fondo).  
- Secciones principales:
  - **Historia** → Contexto y origen del movimiento.  
  - **Merch** → Tienda y productos oficiales.  
  - **Noticias** → Actualidad y novedades.  
  - **Batallas** → Competencias y eventos destacados.  
  - **Ubicación** → Información geográfica y cultural.  
  - **Contacto** → Formulario funcional para comunicación.  

---

## Identidad visual

- **Colores:** Blanco y negro como base, con acentos en rojo para resaltar interacción.  
- **Tipografía:** Urban Font (estilo fuerte y cultural).  
- **Logo:** `halasinfondo.png` integrado en la navbar y en la marca.  
- **Estética:** Minimalista, clara y con foco en la cultura visual urbana.  

---

## Objetivos del proyecto

- Crear un **espacio colaborativo** donde estética, funcionalidad e identidad se fusionen.  
- Documentar la historia y símbolos familiares de Halabalusa como parte de la **marca cultural**.  
- Ofrecer una **landing page responsiva** con estructura semántica clara y accesible.  
- Integrar recursos visuales, tablas comparativas y ejemplos editables para formación y documentación.  
- Promover la **cultura visual y narrativa urbana** en cada detalle del proyecto.
